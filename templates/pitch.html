The Problem

Custom order spec collection is broken

Every custom order on Etsy starts the same way: a seller sends a message asking for details, then waits hours or days for each reply. The same questions, over and over, for every single order.

1

Buyer purchases a custom item. Seller sends a message: "Thanks! I need a few details..."

2

Buyer responds with partial info. "I want it in gold." (But what about the name? The size? The font?)

3

Seller asks follow-up questions. Buyer takes 24 hours to respond to each one.

4

Repeat 4-8 times over several days. Meanwhile, the order sits unstarted and the production queue backs up.
